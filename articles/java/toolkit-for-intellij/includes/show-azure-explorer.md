## <a name="displaying-the-azure-explorer-tool-window"></a>Azure 탐색기 도구 창 표시

IntelliJ에 **Azure 탐색기** 도구 창이 표시되지 않으면 **보기**, **도구 창**, **Azure 탐색기**를 차례로 클릭합니다.
