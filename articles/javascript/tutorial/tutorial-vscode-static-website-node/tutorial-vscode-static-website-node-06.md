## <a name="delete-storage-account"></a>저장소 계정 삭제

스토리지를 만들면 최소한의 진행 비용(스토리지의 경우 매우 작음)이 발생합니다. 이러한 비용을 방지하려면 Visual Studio Code의 **Azure: Storage** 탐색기에서 스토리지 계정을 마우스 오른쪽 단추로 클릭하고 **스토리지 계정 삭제** 를 선택합니다.
