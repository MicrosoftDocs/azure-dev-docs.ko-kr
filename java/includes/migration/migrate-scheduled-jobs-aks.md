AKS 클러스터에서 예약된 작업을 실행하려면 필요에 따라 Kubernetes CronJob을 정의합니다. 자세한 내용은 [CronJob을 사용하여 자동화된 작업 실행](https://kubernetes.io/docs/tasks/job/automated-tasks-with-cron-jobs/)을 참조하세요.
