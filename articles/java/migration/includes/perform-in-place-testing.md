### <a name="perform-in-place-testing"></a>내부 테스트 수행

컨테이너 이미지를 만들기 전에 애플리케이션을 AKS에서 사용하려는 JDK 및 WildFly 버전으로 마이그레이션합니다. 호환성과 성능을 보장하기 위해 애플리케이션을 철저히 테스트합니다.
